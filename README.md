
# ClairCRM

CRM local sans installation : ouvrez simplement `index.html` dans votre navigateur.

Fonctionnalités incluses :

- Tableau de bord et indicateurs clés.
- Ajout, recherche et suppression de contacts.
- Pipeline d’opportunités par étape.
- Création, filtrage et validation de tâches.
- Sauvegarde automatique des données dans le navigateur (`localStorage`).
